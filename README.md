# UTAH REIA Wholesaler Calculator

This project is a web-based calculator designed for real estate wholesalers and investors, specifically tailored for the Utah Real Estate Investors Association (UTAH REIA).

## Features

- Step-by-step interface to calculate Max Allowable Offer (MAO), wholesale fee, and generate professional reports for cash buyers.
- Supports both House Flipper and Rental Investor buyer types.
- Input fields for property details, repair estimates, closing costs, holding costs, sale costs, and buyer profit requirements.
- Generates a detailed analysis report and cost breakdown.
- Downloadable PDF report for sharing with buyers.
- User-friendly validation and error messages for required fields.

## Usage

1. Enter property information and buyer type.
2. Fill in all required financial fields.
3. Click "Finish Analysis" to generate your report.
4. Download or share the generated PDF report.

## Disclaimer

Estimates provided by this calculator are educational only and should not be considered financial advice.
